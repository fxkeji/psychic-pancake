## 🚀 使用方法

### 方法1：直接运行（推荐）
```bash
# 双击运行
智能拼版计算系统（优化版）.html

# 或使用启动脚本
.\启动应用.bat
```

### 方法2：使用部署包
```bash
# 进入部署目录
cd deploy

# 快速启动
.\快速启动.bat
```

### 方法3：构建EXE/APK
```bash
# Windows EXE
.\build.bat

# Android APK
.\apk-build.bat

# 一键构建
.\快速构建.bat
```
